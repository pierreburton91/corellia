# Startup-seed

A startup seed for static website development.

Uses :
* Gulp
* Gulp-stylus
* Gulp-autoprefixer
* Gulp-clean-css
* Gulp-concat
* Gulp-uglify
* Gulp-rename
* Gulp-pug
* Gulp-imagemin
* Browser-sync

Includes :
* gulpfile.js (Pre-configured)
* robots.txt
* sitemap.xml
* humans.txt
* favicon.png
* banners and icon images
* index.pug boilerplate
